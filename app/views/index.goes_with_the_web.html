Pakyow is an open-source framework for building apps that embrace the web, while encouraging a development process that is friendly to both designers and developers. It's built for getting along.

Power to the designer.

Own the Front End

The front end is created in isolation, giving the designer total control over the process. Work in one space from start to finish.

Use Native Templates

Get all the benefits of templating without the template language. Build views in pure HTML; no logic, no new syntax.

Build View First

Create a working prototype of your app without a line of code. Later, add the backend without changing the view.

Software made simple.

Designed for consistency

All parts of the application speak the common language of data, providing a consistent way of thinking about and writing code.

Separates Your Concerns

Roles of designer/developer, view/logic, client/server are clearly defined and maintaind throughout the development process.

Advanced Behavior Built-In

Control the client from the server and update the view in realtime as data changes, all with no added effort. Things just work.

Goes with the web.

The Client is the Interface

The client, together with the code that powers it, define the interface to the application.

Logic Stays Server-Side

Application logic remains on the backend and instructions, not code, are pushed to the client.

The web is embraced

A traditional client-server model is supported, leaving the web to be the web.
