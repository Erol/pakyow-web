Knock out your hardest hitting projects.

Pakyow is a web framework for Ruby that packs a serious punch. Pound for pound it's the best way to build a web app. Get ready to rumble.

    Let views be views.

    Views are 100% HTML, no template language required. The view has finally been freed from logic.

    Views are data aware.

    A view knows what it presents. Use this to create powerful connections between business logic and views.

    Keep moving forward.

    Prototype an app by building the views first. Then write the view logic without changing a single view.

    Getting Started

    The Ruby language must be installed, as well as a package manager, such as RubyGems. Pakyow plays nice with MRI (1.8.7 and 1.9.2) and JRuby. Rubinius support is in the works.

    Installation is super easy: gem install pakyow

    Here's the simplest app you can build:

    require 'rubygems' require 'pakyow-core' class HelloApp < Pakyow::Application routes do get '/' do response.body << "Hello Pakyow" end end end HelloApp.run

    Pakyow includes an application generator that makes getting started even easier. pakyow new application

    Then start the application server: cd application
    $ pakyow server

    That's it! You'll find your application running here: localhost:3000. Check out the manual for more information.
