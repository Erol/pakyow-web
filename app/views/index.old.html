Pakyow is an open-source framework for building performant web apps that embrace the web. Separation of view, logic, and data is maintained, encouraging a development process that is straightforward and friendly to designers and developers. It's simple. 

<article data-scope="post">
  <p data-prop="body">
    Body goes here.
  </p>
</article>

<html>
  <body>
    <section data-container="main">
      Main Content Here
    </section>
  </body>
<html>

Pakyow views are logicless, composable, and aware of the data they represent. No additional markup or tags are required and complete control over the view is placed in the hands of the designer. 

view.scope(:post).apply([
  { body: 'one' },
  { body: 'two' }
])

<html>
  <body>
    <section data-container="main">
      <article data-scope="post">
        <p data-prop="body">
          one
        </p>
      </article>

      <article data-scope="post">
        <p data-prop="body">
          two
        </p>
      </article>
    </section>
  </body>
<html>

Back-end logic makes it easy to transform the view based on data. Because the view is data-aware, data can be bound to the view by reference. At the time of binding, transformations can be applied to the view based on the data being bound.

Adding back-end logic is non-destructive. This means the view layer becomes a sturdy prototype that can be used and extended throughout the development process; all with no added effort.
